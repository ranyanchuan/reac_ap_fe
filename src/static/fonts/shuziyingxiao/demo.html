第一步：使用font-face声明字体 
@font-face {font-family: 'ecmifont';
    src: url('ecmifont.eot'); /* IE9*/
    src: url('ecmifont.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('ecmifont.woff') format('woff'), /* chrome、firefox */
    url('ecmifont.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('ecmifont.svg#ecmifont') format('svg'); /* iOS 4.1- */
}
第二步：定义使用ecmifont的样式 
.ecmifont{
    font-family:"ecmifont" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
第三步：挑选相应图标并获取字体编码，应用于页面 
<i class="ecmifont">&#x33;</i>
